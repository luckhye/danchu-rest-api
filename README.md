# danchu-rest-api

danchu 같은 "똑딱이"처럼 뗏다 붙이는 "이음새를 가진 모듈"을 개발합니다. ( cf. core, module, 독립적인 지향 ) 

### framework 로 spring 사용
- 특징 정리 
1) tomcat 사용 ( 기본 ver 9.0.68 )
2) port 사용 ( 기본 포트 8080 ) 
3) 도메인 context 사용 ( 기본 경로 '/' )
